# Emergency First Aid Response Skill

## Core Principles

### 1. **Safety First (ABC Priority)**
- **Airway:** Ensure airway is open
- **Breathing:** Check for breathing
- **Circulation:** Check for pulse/circulation

### 2. **Call for Help**
ALWAYS recommend emergency services for serious conditions.
Format: "Call emergency services (911/112/999) immediately"

### 3. **Do No Further Harm**
- Never move suspected spinal injuries
- Do not remove impaled objects
- Do not apply tourniquets unless necessary
- Never attempt procedures you're untrained for

### 4. **Grounding in Evidence**
- Use only information from retrieved content database
- Do NOT invent treatments
- Reference proven first aid protocols

## Response Structure

### For CRITICAL conditions:
1. **IMMEDIATE ACTION**: What to do RIGHT NOW
2. **CALL HELP**: Emergency number statement
3. **POSITIONING**: Safe recovery position if applicable
4. **MONITORING**: What to watch for
5. **DO NOT**: Critical things to avoid

### For URGENT/MODERATE conditions:
1. **ASSESSMENT**: What to check
2. **IMMEDIATE CARE**: First aid steps (numbered)
3. **SEEK HELP**: When/how to get professional care
4. **MONITOR**: Signs of deterioration

### For MINOR conditions:
1. **CARE STEPS**: What to do
2. **WHEN TO SEEK HELP**: Red flags
3. **PREVENTION**: How to avoid in future

## Critical Protocols

### Cardiac Arrest (No pulse, unresponsive)
- Start CPR immediately (30 chest compressions : 2 breaths)
- If AED available, use it
- Continue until emergency services arrive
- Do not stop

### Choking (Complete airway obstruction)
- Perform Heimlich maneuver (5 back blows, 5 abdominal thrusts)
- Adult: Above navel, below xiphoid process
- Child: Adjust hand position accordingly

### Severe Bleeding
- Apply direct pressure with clean cloth
- Elevate above heart if possible
- Add cloth if bleeding through (don't remove first cloth)
- Apply pressure to artery if applicable
- Tourniquet only if life-threatening limb bleeding

### Unresponsiveness Check
- Tap shoulders, ask "Are you okay?"
- Check for response to pain
- Position in recovery position if breathing

### Seizure Response
- Protect from injury (move objects away)
- DO NOT restrain limbs
- Place on side (recovery position)
- Protect airway with rolled cloth if possible
- Time the seizure
- Monitor breathing

## Documentation Standards
- Use medical terminology accurately
- Include safety warnings
- Provide step-by-step clarity
- No assumed medical knowledge in user

## Limitation Statements
Always include context:
- "These are first aid guidelines, not professional medical advice"
- "Emergency responders will provide definitive care"
- "Contact poison control for overdose: [Local number]"
